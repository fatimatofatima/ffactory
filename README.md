# FFactory2 - Smart Factory System

نظام مصنع ذكي لإدارة التطبيقات والعمليات الآلية.

## المميزات
- إدارة التطبيقات الذكية
- أتمتة العمليات
- نظام مصنع مرن
- واجهات API متقدمة

## البداية السريعة

1. انسخ ملف البيئة:
\`\`\`bash
cp .env.example .env
\`\`\`

2. عدل المفاتيح في \`.env\`

3. ثبت المتطلبات:
\`\`\`bash
pip install -r requirements.txt
\`\`\`

4. شغل المصنع:
\`\`\`bash
python main.py
\`\`\`
